# 50-Year AI Product Manager Prompt Toolkit

Welcome to the **50-Year AI Product Manager Prompt Toolkit**. This repository contains a curated, hyper-enhanced collection of 20 essential product management prompts designed to run on advanced LLMs (like Gemini, Claude, or GPT-4).

Each prompt in this toolkit has been expanded and polished from the perspective of an veteran Product Leader with decades of experience spanning multiple technology eras. They are designed to elicit deep, rigorous, and actionable insights, avoiding generic AI responses.

## Structure of the Toolkit

Every prompt file in the `prompts/` directory contains:
1. **Strategic Context**: When to use the prompt, key traps to avoid, and the underlying product rationale.
2. **The Hyper-Enhanced Master Prompt**: An optimized, instruction-dense prompt ready to copy and paste.
3. **Follow-Up & Deep-Dive Prompts**: Secondary prompts to drill down into specific areas, stress-test the initial outputs, or generate supplementary deliverables.
4. **50-Year PM Wisdom**: Tactical advice, contrarian viewpoints, and organizational tips on how to apply the findings.

## Quick Start Guide

1. Open the file corresponding to the task you need to complete.
2. Review the **Strategic Context** and **PM Wisdom** sections to understand the key levers for success.
3. Copy the **Primary Prompt**, replace the `[BRACKETED]` placeholders with your product's details, and paste it into your preferred LLM.
4. Use the provided **Follow-Up Prompts** to iterate and refine the output based on what the LLM produces.
